Exercício 06 - Lourdinas

Um sistema de controle de clima inteligente ajusta automaticamente a temperatura de um ambiente conforme dois fatores: a estação do ano e o nível de conforto selecionado pelo usuário. Primeiro, o sistema deve verificar se a estação informada é "verao", "inverno" ou "outono". Depois, dentro de cada estação, a temperatura alvo depende do nível de conforto:

  • Verão
    • "economico" → 26° C
    • "confortavel" → 23° C
  • Inverno
    • "economico" → 20° C
    • "confortavel" → 24° C
  • Outono
    • "economico" → 22° C
    • "confortavel" → 21° C

Se a estação informada não for nenhuma das válidas, o sistema deve retornar: "Estação inválida." Se o nível de conforto não for "economico" nem "confortavel", o sistema deve retornar: "Nível de conforto inválido." Desenvolva, em JavaScript, um código que receba a estação e o nível de conforto e retorne a temperatura configurada pelo sistema.

Exemplos:

  • Entrada: "verao", "confortavel" → Saída: 23° C
  • Entrada: "inverno", "economico" → Saída: 20° C
  • Entrada: "primavera", "confortavel" → Saída: "Estação inválida."
  • Entrada: "outono", "luxo" → Saída: "Nível de conforto inválido."
